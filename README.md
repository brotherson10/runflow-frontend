# RunFlow Frontend

Responsive landing page developed during the DecodeLabs Full Stack Internship.

This project started as a static landing page and evolved into a dynamic frontend integrated with a backend API.

## 🚀 Technologies

* HTML5
* CSS3
* JavaScript (Vanilla)

## ✨ Features

* Responsive layout (mobile-first)
* Semantic HTML structure
* Mobile navigation menu
* Clean and modern UI
* Dynamic data rendering from API
* Runner creation form (connected to backend)

## 🔗 Backend Integration

This frontend consumes data from the RunFlow API:

👉 https://github.com/brotherson10/runflow-backend

Using the Fetch API, the frontend:

* Retrieves runners from the database (GET)
* Displays runners dynamically on the page
* Sends new runners to the backend (POST)

## ⚙️ How to Run

1. Clone this repository:

```bash
git clone https://github.com/brotherson10/runflow-frontend.git
```

2. Open the project folder

3. Run with Live Server or open `index.html` in your browser

4. Make sure the backend is running at:

```text
http://localhost:3000
```
